# Changelog — stop-slop-fr

## 1.0.0

- Première version. Adaptation française de [stop-slop](https://github.com/hardikpandya/stop-slop).
- `SKILL.md` : 8 règles de base, checklist de vérification rapide, grille de notation (en français).
- `references/expressions.md` : amorces creuses, béquilles d'emphase, connecteurs scolaires, mots fétiches, jargon et anglicismes, adverbes en -ment, méta-commentaire, formules de conclusion automatiques.
- `references/structures.md` : faux contrastes, énumération par la négation, gérondif final, phrases clivées, agentivité factice, « on » impersonnel, voix passive, énumérations ternaires, typographie française.
- `references/exemples.md` : 7 transformations avant/après pensées en français.
- Empaqueté comme plugin Claude Code (`.claude-plugin/plugin.json` + `skills/`).
